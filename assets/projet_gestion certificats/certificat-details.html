Détail du projet

Application Flutter Mobile

Gestion des Certificats de Domicile

Application mobile destinée aux collectivités locales du Sénégal, permettant de digitaliser la demande, la validation, et la délivrance des certificats de domicile avec génération PDF et intégration de paiement numérique.

  • + Validation agent :
    Interface sécurisée pour agent administratif.
  • + Certificat PDF :
    Export PDF signé avec cachet numérique.
  • + Historique :
    Suivi complet de toutes les demandes.
  • + Paiement mobile :
    Intégration de PayDunya pour certificat.

Tech Stack

Flutter Dart GetX SQLite PayDunya API PDF + Signature

Mon rôle : Conception, modélisation, développement full Flutter, intégration de l'API de paiement, tests utilisateurs.

Ce que j'ai appris : Intégration d'API tierce, création de PDF dynamiques, design mobile-first, gestion d'état avec GetX, scénarios de tests terrain.

Possibles améliorations : Passage à Firebase pour multi-utilisateurs, tableau de bord analytics, export Excel des certificats, déploiement sur Play Store.
