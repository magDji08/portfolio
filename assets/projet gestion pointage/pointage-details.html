Système de Pointage Intelligent

Solution intégrée combinant RFID, biométrie et plateforme web pour une gestion optimisée des présences

Contexte du projet

Développé dans le cadre de mon mémoire de fin d'études, ce système innovant répond aux besoins des établissements cherchant à automatiser et fiabiliser le suivi des présences. Il combine des technologies matérielles et logicielles pour offrir une solution complète et sécurisée.

Problématique résolue

Remplacement des systèmes de pointage manuels sujets aux erreurs par une solution automatisée, fiable grâce à la biométrie, et offrant une gestion centralisée des données.

Objectifs atteints

  • Authentification multi-facteurs (RFID + biométrie)

  • Transmission des données en temps réel

  • Interface web de gestion centralisée

  • Génération automatisée de rapports

Fonctionnalités principales

Authentification sécurisée

Double méthode d'identification via carte RFID et empreinte digitale pour une sécurité renforcée.

Transmission des données

Communication via port série USB ou Wi-Fi (ESP8266) pour une intégration flexible.

Gestion centralisée

Plateforme web pour administrer utilisateurs, consulter les présences et gérer les accès.

Statistiques avancées

Visualisation des retards, absences et taux de ponctualité pour une meilleure analyse.

Export de rapports

Génération de documents PDF/Excel pour archivage et traitement externe.

Sécurité des données

Stockage sécurisé avec chiffrement des données biométriques sensibles.

Architecture technique

Composants matériels

  • Arduino Uno (microcontrôleur)
  • Module RFID RC522
  • Capteur d'empreinte digitale R305
  • Module Wi-Fi ESP8266 (optionnel)

Stack logicielle

  • Django (framework Python back-end)
  • MySQL (gestion via XAMPP)
  • Tailwind CSS (design front-end)
  • JavaScript (interactivité)
